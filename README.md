# QR Code Generator

A modern, professional QR code generator built with Next.js and React. Create custom QR codes for various purposes with easy-to-use customization options.

## Features

- **Multiple QR Code Types:** Generate QR codes for:
  - Plain Text
  - URLs
  - Email addresses
  - Phone numbers
  - SMS messages
  - WiFi networks

- **Customization Options:**
  - Adjustable size (100-1000px)
  - Margin control (0-10px)
  - Foreground and background colors
  - Error correction levels (L/M/Q/H)

- **Download Formats:**
  - PNG
  - JPEG
  - SVG

## Tech Stack

- **Next.js 16.1.6** - React framework
- **React 19.2.3** - UI library
- **TypeScript** - Type safety
- **Tailwind CSS 4** - Styling
- **Lucide React** - Icons
- **qrcode** - QR code generation library

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open your browser and navigate to `http://localhost:3000`

### Build

```bash
npm run build
```

### Start Production Server

```bash
npm start
```

## Usage

1. **Select QR Code Type:** Choose the type of QR code you want to generate from the dropdown.
2. **Enter Data:** Fill in the required information based on the selected type.
3. **Customize:** Adjust the QR code appearance using the customization options.
4. **Download:** Click the download button to save your QR code in the desired format.
